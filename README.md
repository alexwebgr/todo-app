#Todo App
A simple todo app using rails 6 with image upload and basic todo management.
Users can be added to multiple lists.

### Features
* Ruby 2.6.3
* Rails 6
* Postgres
* Encrypted credentials
* Active Storage
* Webpacker
* jQuery
* Bootstrap 
* Fontawesome icons

### Not included
* User auth
* ACL
* Deployment strategy
* The icons
